Course Overview

Typography is the invisible architect of the web. In this course, we move beyond simple font selection to explore the technical and aesthetic nuances of digital type. You'll learn about vertical rhythm, responsive modular scales, variable fonts, and how to use CSS to create layouts that are not only beautiful but also accessible and performant.

What You'll Learn

Principles of Type Pairing & Hierarchy
Implementing Responsive Modular Scales
Optimizing Type for Web Performance
Variable Fonts & CSS Customization
Accessibility Standards (WCAG) for Type
Visual Storytelling through Typography
